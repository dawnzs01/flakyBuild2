# Roadmap

Arctic 预计每2-3个月发布一个大版本，每1个月发布一个小版本，每个版本都会有一个发布计划，发布计划会在 [Github Milestones](https://github.com/NetEase/arctic/milestones) 中发布。

## 2022年发布计划

| 版本                                                     | 发布日期          |
| --------------------------------------------------------| ---------------- |
| [0.3.1](https://github.com/NetEase/arctic/milestone/1)  | 2022.08          |
